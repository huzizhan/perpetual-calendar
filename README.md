# 📅 万年历 (Perpetual Calendar)

终端版万年历，支持公历/农历/节气/节日显示，纯 Python 实现，零额外二进制依赖。

## 功能特性

- **公历 + 农历** — 每日同时显示公历日期与农历日期（支持闰月）
- **天干地支** — 年份的干支纪年（如丙午年）+ 生肖（如马年）
- **二十四节气** — 在日历上标注当月节气（芒种、夏至…）
- **传统节日** — 春节、元宵、端午、中秋等自动标注
- **公历节日** — 元旦、国庆、圣诞等
- **周末高亮** — 周六蓝色、周日红色
- **今天标记** — 绿色高亮当天
- **交互模式** — 方向键翻月翻年、快捷键跳转
- **全年视图** — 3×4 网格一次性展示全年

## 环境要求

- Python 3.10+
- 终端需支持 Unicode / True Color（macOS Terminal、iTerm2、Windows Terminal 均支持）

## 快速开始

```bash
# 安装依赖
pip install -r requirements.txt

# 显示当月日历
python main.py

# 显示 2026 年全年
python main.py 2026

# 显示 2026 年 6 月
python main.py 2026 6

# 交互模式（方向键导航）
python main.py -i
```

## 交互模式快捷键

| 按键 | 功能 |
|------|------|
| `←` `→` / `h` `l` | 上一月 / 下一月 |
| `↑` `↓` / `k` `j` | 上一年 / 下一年 |
| `t` | 回到今天 |
| `y` | 输入年份跳转 |
| `m` | 输入月份跳转 |
| `a` | 打印全年视图 |
| `q` / `ESC` | 退出 |

## 项目结构

```
perpetual-calendar/
├── main.py                # 主入口（CLI + 参数解析）
├── requirements.txt       # 依赖清单
├── README.md
└── src/
    ├── lunar.py           # 农历转换引擎（内嵌 1900-2100 年数据）
    ├── solar_terms.py     # 二十四节气计算
    ├── renderer.py        # Rich 终端渲染器
    └── tui.py             # 交互式 TUI
```

## 技术说明

- 农历数据内嵌 1900–2100 年完整编码表，**无需联网**、**无需额外数据文件**
- 节气计算基于简化天文算法，精度 ±1 天（日常使用完全足够）
- 终端渲染使用 [Rich](https://github.com/Textualize/rich) 库
